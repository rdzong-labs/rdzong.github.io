Getting Started

Running DEC Potala on Raspberry Pi

Potala is our nano version of DEC Platform for deployment on single board computer (SBC) such as Raspberry Pi.

Prerequisites

  • Raspberry Pi 3 Model B+, or Pi 4 Model

  • Picamera Module v2.1, Sony IMX219 8-megapixel sensor

  • [Optional] USB storage: flash drive, SSD drive, or hard drive

Installation

If you are not familiar with Raspberry Pi (RPi), please visit Getting started with Raspberry Pi to get acquaintance with your small computer.

We have tested Potala on Raspbian OS stretch and buster. If your RPi has older version, please follow the following instructions to upgrade the OS. Ideally, you should erase and install a fresh OS on the micro SD card. However, if you are an adventurer please follow the official guidelines, Updating and upgrading Raspbian.

To clean install the latest Raspbian OS:

  • From a computer or laptop, download the Raspbian OS from Raspberry Pi download

    • Select “Raspbian“
    • Choose the “desktop” version; for example, “Raspbian Buster with desktop”. We are using the desktop version that comes with the GUI, so to demonstrate the camera streaming on the monitor connected with HDMI cable.
      Otherwsie, Potala will work with the “Lite” version as well, for example, “Raspbian Buster Lite”
    • Click “Download ZIP”
    • For earlier version, Raspbian Stretch, go to downloads archive

  • From the computer and use the downloaded .zip file, install a Raspberry Pi operating system image on an SD card; also described in Installing operating system images:

    • check the SD card requirements but if your SD card is not greater than 32GB, you should be OK to proceed
    • Download balenaEtcher and install on your computer; balenaEtcher is a graphical tool to install OS to SD card that works on Mac OS, Linux and Windows.
    • Insert the SD card into the corresponding slot on your computer
    • Follow the Installing operating system images

  • While the SD card is being flashed, prepare your Raspberry Pi BUT don’t connect the power supply yet. Connect the HDMI cable to a monitor, mouse & keyboard to USB port, get the wifi SSID and password ready, and setup the camera. For instructions on how to setup the camera, please visit Getting started with the Camera Module.

  • When the SD card is flashed, eject from computer & insert it into the Rapsberry Pi to start first booting.

  • Continue to the next step, “Setup Raspberry Pi”

Setup Raspberry Pi

  • After first boot, follow the screen instructions to setup the OS

  • Login using the login id/password you provided during OS setup, a step above. The default login id: pi; and password: raspberry.

  • If any of the setup is incorrect, reconfigure using raspi-config from terminal, setup that includes: change password, setup country, language, timezone, keyboard, wifi, etc.

    sudo raspi-config

    Note: to reconfigure using GUI, go to Start - Preference - Raspberry Pi Configuration

  • Assuming you have setup the camera while waiting for the SD to be flashed, proceed with enabling the camera on the next step; otherwise follow the instructions on how to setup the camera, by visiting Getting started with the Camera Module

  • If not already done, enable the camera using 'raspi-config from the terminal, or from the Raspberry Pi Configuration GUI. For more detailed instructions, please visit Connect the Camera Module

  • check if the camera is mounted correctly and working, by opening a terminal and type:

    cd ~
raspistill -v -o test.jpg

    Refer to Raspberry Camera Module

  • Finally, update the raspbian OS to get the latest patches

    sudo apt-get update
sudo apt-get dist-upgrade

  • Now your RPi host is ready for AI vision applications

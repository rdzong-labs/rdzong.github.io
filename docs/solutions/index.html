AI Solutions

Test drive Potala Home Monitoring system

Potala Home Monitoring system is a home surveillance system with AI-based object recognition.

Potala Home Monitoring system can be configured to capture image frame based on two different modes: manual, or motion triggerred. Captured images are processed in offline batch mode for recognizing object(s) in the picture. There is an additional mode that capture images, also based on motion triggered, and process object recognition inline albeit it is slower.

Prerequisites

  • Raspberry Pi 3 Model B+, or Pi 4 Model

  • Picamera Module v2.1, Sony IMX219 8-megapixel sensor

  • [Optional] USB storage: flash drive, SSD drive, or hard drive

  • Docker Container Runtime; otherwise please refer to Getting Started

Functional components

The high level functional setup of the home monitoring system is illustrated below.

The hardware components consists of: Raspberry Pi, and the Pi camera. It’s assumed that the RPi is connected to a monitor, keyboard and mouse, at least during initial setup; they can be removed after completion of the setup. Future access to RPi can be done using SSH for terminal access, and VNC for GUI access. Both of these services need to be enabled because they are not enabled by default. Please refer to Remote Access - SSH and Remote Access - VNC for how to enable these services.

The software components (services) consists of:

  • Image web portal to view and review captured images;m

  • Camera monitoring services

Overview

Dzong Edge Computing Platform (DEC) is a micro-service Edge Computing platform that can accelerates the time-to-market of your AI software.

What is it?

Artifical Intelligence is generating high demands for intelligent software that can senses the physical environment, and uses the real time data to make in-situ decisions instantly. To meet this demands the software needs to be deployed closer to the source of actions, i.e., the EDGE.

Edge Computing can benefit from many capabilities similar the cloud but will also need a new set of features specifically for operating at the edge of the network. That is what Dzong Edge Computing Platform (DEC) offers.

Why do I want it?

Development of intelligent software is complex, time consuming, and requires integration with the embedded system (i.e., hardware) it’s deployed on. The hardware architecture varies making the effort for software reuse more tedious.

Dzong Edge Computing Platform (DEC) enables microservices architecture (MSA) applications and services, and built with the following capabilities:

  • support development of microservices application; application that is designed as a collection of loosely coupled services, fine-grained and lightweight integration protocol
  • collections reusable and extesible software stacks and service components, including those for development AI software. These components simplify development and reduce development time;
  • registry-based deployment that enables cross-functional team collaboration, and streamline deployment to minimize frictions in promoting software from development to QA to production
  • For your operational team, robust and resilience platform, scaling on-demand, and equipped with monitoring, notification and troubleshooting tools

The DEC platform provides end-to-end capabilities for your edge computing and therefore enabling you to accelerate time-to-market of your AI software with confidence.

The platform has been tested on Single board computer (SBC): Raspberry Pi, and Jetson Nano for AI vision application. See AI Solutions. We are also working on additional features to support AI applications and on scaling to clusters of SBC.
